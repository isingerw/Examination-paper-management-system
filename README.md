# 试卷管理系统

#### 介绍
J2EE架构与程序设计实践，第七组：试卷管理系统（Examination paper management system），需求功能：题型管理、试题管理、试题选项管理、用户答题记录、用户得分管理、试卷管理、错题集。
项目成员：方圆（项目经理）、张欣（前后端开发）、林东（开发人员）、黎国明（开发人员）、刘浩（数据库开发）、林光耀（系统测试）。
